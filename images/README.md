# Images Directory

**Action Required:** Copy the PNG image files from the main branch to this directory.

## Quick Setup

From the repository root, run:
```bash
./copy-images-from-main.sh
```

OR manually:
```bash
git checkout main
cp *.png images/
git checkout copilot/create-guide-for-machine
```
